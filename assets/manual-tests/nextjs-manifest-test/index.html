Next.js Endpoint Discovery Test Page

This page simulates a Next.js application to test the endpoint discovery feature. It requires a local web server to work correctly.

Expected Findings: The extension should discover the following 5 endpoints from the manifest file. It should automatically filter out internal routes like /_app.

  • /
  • /404
  • /about-us
  • /admin/dashboard
  • /products/[slug]

The source for each finding should be the full URL to the manifest file, e.g., http://localhost:8000/_next/static/fake-build-id-12345/_buildManifest.js.
